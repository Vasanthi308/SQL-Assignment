# SQL-Assignment

This repository contains an SQL assignment project designed to practice and demonstrate essential SQL skills used in real-world data analysis. The project includes SQL queries covering data filtering using WHERE, aggregations using COUNT, SUM, and AVG, grouping with GROUP BY and HAVING, JOIN operations across multiple tables, subqueries, and conditional logic.

The assignment showcases my ability to write structured, efficient, and readable SQL queries while solving practical data problems using relational databases.
